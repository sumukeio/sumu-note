# 表格删除列图标设计方案

## 当前设计
- **图标**: `X` (lucide-react)
- **样式**: 红色圆形背景，白色 X 图标
- **大小**: 4x4 (w-4 h-4)
- **问题**: 用户反馈图标太丑

---

## 设计方案选择

### 方案 A: Minus 图标（已实现）
- **图标**: `Minus` (lucide-react)
- **样式**: 红色圆形背景，白色横线图标
- **大小**: 5x5 (w-5 h-5)
- **视觉效果**: 简洁，表示"移除"的含义
- **优点**: 
  - 简洁明了
  - 符合常见的删除/移除设计模式
  - 与删除行按钮保持一致
- **缺点**: 
  - 可能不够直观（横线可能被理解为"减号"）

**代码示例**:
```tsx
<button className="absolute -top-1 -right-1 w-5 h-5 bg-red-500 hover:bg-red-600 text-white rounded-full opacity-0 group-hover/col:opacity-100 transition-opacity flex items-center justify-center shadow-sm">
  <Minus className="w-3 h-3" />
</button>
```

---

### 方案 B: XCircle 图标
- **图标**: `XCircle` (lucide-react)
- **样式**: 红色圆形背景，白色 X 图标（带圆圈）
- **大小**: 5x5 (w-5 h-5)
- **视觉效果**: 更明显的删除标识
- **优点**: 
  - 删除意图更明显
  - 图标更精致
- **缺点**: 
  - 可能显得过于突出
  - 与圆形背景可能重复

**代码示例**:
```tsx
<button className="absolute -top-1 -right-1 w-5 h-5 bg-red-500 hover:bg-red-600 text-white rounded-full opacity-0 group-hover/col:opacity-100 transition-opacity flex items-center justify-center shadow-sm">
  <XCircle className="w-3 h-3" />
</button>
```

---

### 方案 C: Trash2 图标（小尺寸）
- **图标**: `Trash2` (lucide-react)
- **样式**: 红色圆形背景，白色垃圾桶图标
- **大小**: 5x5 (w-5 h-5)
- **视觉效果**: 直观的删除含义
- **优点**: 
  - 删除意图最明显
  - 用户熟悉垃圾桶图标
- **缺点**: 
  - 在小尺寸下可能不够清晰
  - 与删除表格按钮重复（但尺寸不同）

**代码示例**:
```tsx
<button className="absolute -top-1 -right-1 w-5 h-5 bg-red-500 hover:bg-red-600 text-white rounded-full opacity-0 group-hover/col:opacity-100 transition-opacity flex items-center justify-center shadow-sm">
  <Trash2 className="w-3 h-3" />
</button>
```

---

### 方案 D: 无背景的红色 X
- **图标**: `X` (lucide-react)
- **样式**: 无背景，红色 X 图标
- **大小**: 5x5 (w-5 h-5)
- **视觉效果**: 更轻量，不遮挡内容
- **优点**: 
  - 更轻量，不遮挡表格内容
  - 视觉干扰更小
- **缺点**: 
  - 可能不够明显
  - 与背景对比度可能不够

**代码示例**:
```tsx
<button className="absolute -top-1 -right-1 w-5 h-5 text-red-500 hover:text-red-600 opacity-0 group-hover/col:opacity-100 transition-opacity flex items-center justify-center">
  <X className="w-3 h-3" />
</button>
```

---

### 方案 E: 带边框的红色圆形
- **图标**: `Minus` 或 `X`
- **样式**: 红色边框圆形，白色背景，红色图标
- **大小**: 5x5 (w-5 h-5)
- **视觉效果**: 更精致，与背景融合更好
- **优点**: 
  - 视觉更精致
  - 与表格背景融合更好
- **缺点**: 
  - 对比度可能不够
  - 实现稍复杂

**代码示例**:
```tsx
<button className="absolute -top-1 -right-1 w-5 h-5 bg-white border-2 border-red-500 hover:border-red-600 text-red-500 hover:text-red-600 rounded-full opacity-0 group-hover/col:opacity-100 transition-opacity flex items-center justify-center shadow-sm">
  <Minus className="w-3 h-3" />
</button>
```

## 切换方法

如果需要切换到其他方案，只需修改 `src/components/SegmentedEditor.tsx` 中的图标导入和按钮代码：

1. 修改导入: `import { Plus, X, Trash2, Minus, XCircle } from "lucide-react";`
2. 修改按钮中的图标组件: 将 `<Minus />` 替换为其他图标
3. 根据需要调整样式类名
